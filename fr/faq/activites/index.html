Créer et partager des protocoles d'expériences

← Toutes les FAQ FizziQ App Pédagogie Guide des activités
Dans l'onglet Activités, appuyez sur + pour créer une nouvelle activité. Donnez un titre, un objectif, puis ajoutez les étapes du protocole. Chaque étape guide l'élève dans la réalisation de l'expérience. Un QR code est généré automatiquement pour partager l'activité.

→ Voir le guide complet
Une fois l'activité créée, un QR code est généré. Affichez-le au tableau ou intégrez-le dans un document. Les élèves scannent ce code avec FizziQ pour charger l'activité sur leur appareil. Vous pouvez aussi partager le lien par email ou messagerie.
Utilisez un générateur de QR code en ligne. Le format FizziQ est :
Fizziq//Titre//Objectif//Étape 1//Étape 2//...

Exemple : Fizziq//Le son//Étude de la fréquence//Choisissez un instrument//Mesurez la fréquence

Limite recommandée : 2000 caractères maximum.
FizziQ propose une bibliothèque de plus de 100 protocoles prêts à l'emploi sur le site fizziq.org/activites. Ces activités couvrent physique, chimie, SVT et sont classées par niveau (collège, lycée). Scannez le QR code pour les charger directement.
Dans l'onglet Activités, sélectionnez l'activité à modifier. Appuyez sur l'icône de modification (crayon). Vous pouvez changer le titre, l'objectif et les étapes. Un nouveau QR code sera généré avec les modifications.
