Questions de base

FizziQ Web est une application web gratuite qui transforme votre ordinateur en laboratoire de sciences. Elle permet d'analyser des sons et des videos, d'utiliser des simulations de physique, de connecter des capteurs externes (FizziQ Connect, Arduino, micro:bit) et de creer des rapports scientifiques complets.

FizziQ Web accompagne les eleves dans toute leur demarche scientifique : Experimenter (capturer des donnees), Analyser (organiser et comprendre) et Partager (communiquer les resultats).
Ouvrez simplement votre navigateur et allez sur fizziqweb.web.app. Aucune installation n'est requise, aucun compte a creer. L'application se charge directement dans le navigateur.
Chrome ou Edge sont recommandes pour acceder a toutes les fonctionnalites, notamment la connexion Bluetooth avec FizziQ Connect et la connexion USB avec Arduino/micro:bit.

Firefox et Safari supportent l'analyse audio, l'analyse video et les simulations, mais pas les connexions Bluetooth et USB.

FizziQ mobile utilise les capteurs integres du smartphone (accelerometre, GPS, magnetometre...). FizziQ Web n'a pas acces a ces capteurs mais offre :
  • Analyse audio avancee (spectrogramme haute resolution)
  • Analyse video complete (pointage cinematique)
  • Simulations de physique (pendule, ressort, circuits...)
  • Editeur de rapport avec formules LaTeX
  • Ecran plus grand pour l'analyse
Les deux applications sont complementaires et peuvent echanger des donnees via QR Code.
FizziQ Web necessite une connexion internet pour le chargement initial. Une fois charge, certaines fonctionnalites peuvent fonctionner hors ligne, mais il est recommande de rester connecte pour une experience optimale.

Experimenter - Capturer des donnees

  1. Cliquez sur Experimenter → Analyse audio
  2. Autorisez l'acces au microphone si demande
  3. Choisissez le mode d'affichage : oscillogramme, spectre ou spectrogramme
  4. Appuyez sur Enregistrer pour capturer les donnees
Vous pouvez aussi importer un fichier audio existant.
  1. Cliquez sur Experimenter → Analyse video
  2. Importez une video depuis votre ordinateur
  3. Definissez l'echelle avec une reference connue
  4. Pointez la position de l'objet image par image
FizziQ Web calcule automatiquement les positions, vitesses et accelerations.
Cliquez sur Experimenter → Chronophotographie, puis importez votre image. Vous pouvez aussi creer une chronophotographie a partir d'une video en utilisant l'outil Outils → Video chronophotographie.
FizziQ Web propose 10 simulations de physique :
  • Pendule : oscillations, periode, amortissement
  • Ressort : oscillations harmoniques
  • Balistique : trajectoire de projectiles
  • Plan incline : mouvement avec frottement
  • Centrifugeuse : forces centrifuges
  • Circuit electrique : circuits RC, RL, RLC
  • Ondes sur un lac : propagation, interferences
  • Gaz parfait : thermodynamique PV=nRT
  • Optique : lentilles et miroirs
  • Trois corps : gravitation
Toutes les simulations generent des donnees exploitables comme une vraie experience.
  1. Utilisez Chrome ou Edge
  2. Allumez votre boitier FizziQ Connect
  3. Cliquez sur Experimenter → Capteurs externes
  4. Cliquez sur Bluetooth
  5. Selectionnez votre boitier dans la liste
Les capteurs connectes apparaissent automatiquement.
  1. Programmez votre microcontroleur pour envoyer des donnees au format FZ0:capteur:valeur
  2. Branchez-le en USB avec un cable de donnees
  3. Fermez l'IDE Arduino/MakeCode
  4. Cliquez sur Experimenter → Capteurs externes → USB Serial
  5. Selectionnez le port USB

Analyser - Tableaux et graphiques

Ouvrez un tableau de donnees dans le cahier, puis cliquez sur l'onglet Graphique. Selectionnez les colonnes pour l'axe X et l'axe Y. Vous pouvez afficher jusqu'a 5 courbes simultanement et choisir parmi 6 types d'interpolation.
Dans un tableau, cliquez sur + nouvelle grandeur. Donnez un nom a la colonne et entrez une formule commencant par =. Par exemple : =Distance/Temps pour calculer une vitesse.
FizziQ Web supporte :
  • Operateurs : +, -, *, /, ^ (puissance)
  • Fonctions : sin, cos, tan, sqrt, ln, log, exp, abs
  • Constantes : pi, e
  • Statistiques : somme(), moyenne(), ecartype()
  • Derivation : derivee(), derivee2()
Utilisez les noms de colonnes comme variables : =0.5*masse*vitesse^2
Dans l'onglet Graphique, cliquez sur Interpolation et choisissez le type : lineaire, quadratique, exponentielle, sinusoidale... FizziQ Web calcule automatiquement les coefficients et affiche l'equation.

Partager - Rapport et export

  1. Enrichissez votre cahier avec du texte et des photos
  2. Cliquez sur Editer le rapport
  3. Ajoutez du texte, des images, des formules
  4. Inserez les observations du cahier (tableaux et graphiques)
  5. Cliquez sur Finaliser
Dans l'editeur de rapport, cliquez sur le bouton Σ ou tapez directement entre dollars : $E = mc^2$. Exemples utiles :
  • $\frac{a}{b}$ pour une fraction
  • $\sqrt{x}$ pour une racine carree
  • $x^2$ pour un exposant
  • $\Delta t$ pour le symbole delta
Une fois le rapport finalise, cliquez sur Exporter en PDF ou Exporter en Word. Le fichier est telecharge sur votre ordinateur. Le PDF est ideal pour l'impression, le Word pour des modifications ulterieures.
Exportez une observation en QR Code. Ouvrez FizziQ sur votre smartphone et scannez le QR code pour recuperer les donnees. C'est pratique pour transferer des mesures entre ordinateur et telephone.

Configuration avancee

Le mode restreint permet de n'afficher que les outils necessaires a une activite :
  1. Allez dans Reglages → Personnalisation
  2. Activez le Mode personnalise
  3. Cochez uniquement les outils utiles (ex: Analyse Video, Tableur, Rapport)
Les eleves ne voient que ces outils et ne sont pas distraits par les autres.
Apres avoir configure le mode restreint :
  1. Cliquez sur Generer un lien
  2. Copiez le lien
  3. Partagez-le avec vos eleves (email, ENT, QR code)
Quand les eleves ouvrent le lien, FizziQ Web se charge automatiquement avec votre configuration. C'est ideal pour preparer une seance de TP.
