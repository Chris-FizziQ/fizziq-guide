Questions fréquentes sur l'application mobile iOS et Android

FizziQ est une application gratuite (iOS et Android) qui transforme votre smartphone en laboratoire de sciences. Elle accède aux capteurs intégrés de votre téléphone : accéléromètre, microphone, GPS, magnétomètre, capteur de lumière, etc.

Pour accéder aux capteurs :
  1. Appuyez sur l'onglet Mesures en bas de l'écran
  2. Appuyez sur le bouton Instruments (icône +)
  3. Parcourez les catégories : Mouvement, Son, Lumière, Environnement, etc.
  4. Sélectionnez le capteur souhaité pour commencer vos mesures
Vous avez accès à plus de 50 instruments différents. Pour ajouter des capteurs supplémentaires (température, CO2, pH...), connectez un boîtier FizziQ Connect.

→ Voir le guide complet sur les instruments
Pour enregistrer des mesures, sélectionnez d'abord un instrument, puis appuyez sur le bouton REC (rouge) pour démarrer l'enregistrement. Le bouton devient un carré pendant l'enregistrement. Appuyez à nouveau pour arrêter. Vos données sont automatiquement sauvegardées dans le cahier d'expériences.

→ Voir le guide complet sur l'enregistrement
Le cahier d'expériences est l'espace central de FizziQ où vous organisez vos travaux scientifiques. Il contient vos mesures sous forme de graphiques, tableaux, photos et textes. Vous pouvez créer plusieurs expériences, les renommer, et les exporter en PDF ou CSV.

→ Voir le guide complet sur le cahier
L'analyse cinématique permet d'étudier le mouvement d'objets à partir d'une vidéo. Allez dans Outils → Cinématique, importez ou filmez une vidéo, définissez l'échelle avec une référence connue, puis pointez la position de l'objet image par image. FizziQ calcule automatiquement position, vitesse et accélération.

→ Voir le guide complet sur la cinématique
Pour exporter en CSV, ouvrez votre expérience dans le cahier, appuyez sur l'icône de partage, puis sélectionnez Exporter en CSV. Le fichier contient toutes vos mesures avec horodatage, prêt à être ouvert dans Excel, Google Sheets ou tout logiciel d'analyse.

→ Voir le guide complet sur l'export
Pour générer un rapport PDF, ouvrez votre expérience, appuyez sur l'icône de partage et choisissez Exporter en PDF. Le PDF inclut tous vos graphiques, tableaux, photos et textes explicatifs. Parfait pour rendre un compte-rendu de TP ou documenter une expérience.

→ Voir le guide complet sur l'export
Le mode Duo permet d'afficher deux capteurs en même temps sur l'écran. Dans l'écran de mesure, appuyez sur l'icône Duo (deux carrés). Sélectionnez un second instrument. Les deux mesures s'affichent côte à côte et peuvent être enregistrées simultanément. Idéal pour corréler deux grandeurs physiques.
AskFizziQ est l'assistant IA intégré à FizziQ. Il répond à vos questions sur les sciences et vous aide à interpréter vos mesures. Accédez-y via Outils → AskFizziQ. Posez une question en langage naturel et recevez une explication adaptée à votre niveau. Nécessite une connexion internet.

→ Voir le guide complet sur AskFizziQ
Pour créer un tableau de données, allez dans le cahier d'expériences, appuyez sur + et sélectionnez Tableau. Vous pouvez entrer des valeurs manuellement ou importer des données depuis vos enregistrements. Le tableau peut être modifié, trié et exporté.

→ Voir le guide complet sur les tableaux
Les déclencheurs permettent de démarrer ou arrêter l'enregistrement automatiquement selon un seuil. Dans Outils → Déclencheurs, définissez une condition (ex: son > 60 dB). L'enregistrement démarre quand la condition est remplie. Parfait pour capturer des événements brefs ou imprévisibles.

→ Voir le guide complet sur les outils
Dans le cahier d'expériences, appuyez sur + puis choisissez Photo pour prendre ou importer une image, ou Texte pour ajouter des observations et conclusions. Vous pouvez annoter les photos et formater le texte. Ces éléments seront inclus dans l'export PDF.

→ Voir le guide complet
Dans le cahier, appuyez sur un graphique pour l'ouvrir en plein écran. Utilisez le zoom (pincement) pour examiner les détails. Appuyez sur un point pour voir ses coordonnées exactes. Vous pouvez aussi ajuster les axes, ajouter une courbe de tendance, et mesurer des intervalles.
