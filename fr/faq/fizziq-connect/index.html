FAQ FizziQ Connect

Questions fréquentes sur le boîtier de capteurs externes

← Toutes les FAQ FizziQ App Capteurs Guide complet
Pourquoi FizziQ Connect ? L'application FizziQ utilise les capteurs intégrés de votre smartphone (accéléromètre, microphone, GPS...), mais un smartphone ne peut pas mesurer la température, le CO2, le pH ou la couleur de la lumière. FizziQ Connect comble cette lacune.

Comment ça marche ? C'est un petit boîtier électronique (basé sur le M5 Stack) qui se connecte à votre smartphone en Bluetooth. Vous branchez un capteur externe (température, pH, CO2...) sur le boîtier, et les mesures apparaissent directement dans l'application FizziQ.

Caractéristiques :
  • Connexion Bluetooth Low Energy (portée ~10 m)
  • Écran intégré pour visualiser les mesures
  • Batterie rechargeable (4-8h d'autonomie)
  • Compatible avec plus de 20 capteurs différents
  • Peut fonctionner avec ou sans smartphone

Le coffret contient : le boîtier M5 Stack avec écran, un câble USB-C pour la recharge, un capteur Environnement (température, humidité, pression) et un guide de démarrage rapide. D'autres capteurs peuvent être achetés séparément.

Pour allumer, appuyez sur le bouton latéral gauche pendant 2 secondes. L'écran affiche le logo FizziQ. Pour éteindre, maintenez le même bouton pendant 6 secondes. Le boîtier peut aussi s'éteindre automatiquement après une période d'inactivité.

1) Allumez FizziQ Connect
2) Dans l'application FizziQ, allez dans Instruments → FizziQ Connect
3) Activez le Bluetooth si demandé
4) Sélectionnez votre boîtier dans la liste
Les capteurs apparaissent automatiquement dans FizziQ.

FizziQ Connect est compatible avec trois types de capteurs :
• Capteurs I2C (M5 Stack, Grove, Seeed Studio)
• Capteurs analogiques (0-3.3V)
• Capteurs 1-Wire (ex: sonde DS18B20)
Les capteurs se branchent sur les ports Grove du boîtier.

→ Voir le guide complet
L'autonomie varie selon les capteurs : environ 8 heures avec le capteur Environnement, 4 heures avec le capteur e-CO2. Pour prolonger l'autonomie, activez le mode Auto-off ou réduisez la luminosité de l'écran.

→ Voir le guide complet
Rechargez avec le câble USB-C fourni, branché sur un chargeur USB ou un ordinateur. L'icône de batterie indique le niveau de charge. Une charge complète prend environ 2 heures. Vous pouvez utiliser le boîtier pendant la charge.
Le mode Diffusion permet de partager les mesures du boîtier avec plusieurs appareils FizziQ simultanément. Idéal en classe : un seul capteur peut être visualisé par tous les élèves sur leurs smartphones. Activez ce mode dans les paramètres du boîtier.

Oui, FizziQ Connect peut fonctionner de manière totalement autonome. L'écran intégré affiche les mesures en temps réel, et vous pouvez enregistrer des données dans la mémoire du boîtier. La connexion Bluetooth est optionnelle.
Oui, FizziQ Connect fonctionne avec FizziQ Web sur ordinateur. L'ordinateur doit être équipé de Bluetooth. Sinon, utilisez un dongle Bluetooth USB. La connexion se fait de la même manière que sur smartphone.
Si un capteur n'est pas détecté :
1) Vérifiez que le câble Grove est bien branché des deux côtés
2) Redémarrez le boîtier
3) Vérifiez que le capteur est compatible (I2C, analogique ou 1-Wire)
4) Essayez un autre port ou câble
Certains capteurs nécessitent une mise à jour du firmware.

→ Voir le guide de dépannage
Les mises à jour apportent de nouveaux capteurs et corrections. Connectez le boîtier à un ordinateur via USB-C, puis utilisez l'outil de mise à jour sur le site FizziQ. Suivez les instructions à l'écran. Ne débranchez pas pendant la mise à jour.
